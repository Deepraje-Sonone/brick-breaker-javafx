# Brick Breaker Game 🎮

A classic Brick Breaker game built using JavaFX and Maven.

## Features

- Paddle controlled with keyboard
- Ball collision detection
- Breakable bricks
- Score system
- Lives system
- Game Over screen
- Restart game using R key
- Win condition

## Technologies Used

- Java 25
- JavaFX 25
- Maven
- VS Code

## Controls

| Key | Action |
|------|--------|
| ← | Move Left |
| → | Move Right |
| R | Restart Game |

## Project Structure

```
brickbreaker
│
├── pom.xml
```

## How to Run

```bash
mvn javafx:run
```
